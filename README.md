## 📖 **О боте**

**AnimeCards** — это Telegram-бот, созданный для **развлечения** и **коллекционирования карточек** с аниме персонажами.  

✨ Проект вдохновлён играми *HoYoverse*  и ботом **[AniCard](https://t.me/anicardplaybot)**  

---

## 🎮 **Основные механики**

### 🃏 **Открытие карточек (Gacha)**

- 🎴 В игре доступно **5 редкостей**
- 🎲 Карты выпадают с **вероятностями**
- 📈 Реализована система **soft pity**

После **75-го открытия** шанс на редкость **Хроно** увеличивается:

```python
base + (1 - base) * (x ** 2)
x = (pity - soft_start) / (max_pity - soft_start)
```

📌 **Особенности:**
- 🆕 *Новая карта* → добавляется в инвентарь + валюта  
- 🔁 *Дубликат* → только валюта  

---

### ⚔️ **PvP система**

- 🧩 Колода из **5 карт разных редкостей**
- 🤝 Автоподбор противника
- 💥 Побеждает игрок с **большей суммарной стоимостью**

🏆 **Награды:**
- +1 победа в статистику  
- 🥇 В конце сезона:
  - **Топ-1** получает *лимитированную карту*
